# Project Overview

This project is a no-code digital directory system designed to collect, manage, and display structured data using modern no-code tools.

The system connects Google Sheets, Airtable, and Softr to create a seamless workflow from data entry to user-facing interface.

## Problem
Organizations often struggle with fragmented data sources and lack of structured directories.

## Solution
This project builds a unified system that transforms raw data into a structured and searchable digital directory.

## Outcome
A fully functional, scalable no-code directory system.
