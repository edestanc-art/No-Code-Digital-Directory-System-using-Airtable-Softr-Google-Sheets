# 📊 No-Code Digital Directory System

## 🚀 Overview
This project demonstrates the development of a scalable **no-code digital directory system** using **Google Sheets**, **Airtable**, and **Softr**.  

The system is designed to simplify **data collection, structuring, and presentation** by connecting multiple no-code tools into a unified workflow.

It serves as a practical example of how businesses can build **data-driven digital products without traditional coding**.



## 🎯 Objectives
- Build a centralized data collection system
- Transform raw data into structured relational databases
- Create a user-friendly directory interface
- Demonstrate integration between multiple no-code tools



## 🛠️ Tools Used

### 📄 Google Sheets
- Used as the initial data collection layer
- Enabled fast input and collaboration

### 🗄️ Airtable
- Served as the relational database
- Managed structured and linked datasets
- Enabled filtering, sorting, and data validation

### 🌐 Softr
- Used to build the frontend directory interface
- Converted Airtable data into a live web application
- Provided search and filtering functionality


## 🧱 System Architecture

1. Users input raw data into **Google Sheets**
2. Data is cleaned and structured into **Airtable**
3. Airtable acts as the backend database
4. **Softr** connects to Airtable via API
5. Data is displayed in a searchable digital directory



## 🔄 Data Flow Diagram

Google Sheets → Airtable → Softr Web App → End Users



## 📊 Features
