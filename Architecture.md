# System Architecture

## Components

### 1. Google Sheets
- Data entry point
- Collaborative input system

### 2. Airtable
- Central database
- Data validation and relationships

### 3. Softr
- Frontend web application
- Displays Airtable data dynamically

## Workflow

Google Sheets → Airtable → Softr → Users

## Architecture Type
No-code API-based integration system
